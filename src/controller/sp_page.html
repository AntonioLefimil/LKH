SUB PROBLEM TOUR FILE:

Especifica el nombre de un archivo que contiene un tour que debe utilizarse para dividir el problema original en subproblemas. El número aproximado de nodos en cada uno viene dado por SUBPROBLEM_SIZE. El recorrido está dada por una lista de enteros que proporcionan la secuencia en la que los nodos se visitan en el tour. El recorrido termina con un -1
